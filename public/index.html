PORTFOLIO


PLANT.ME

Plant.me is my capstone project project for Actualize. The app builds a schedule for users to help keep on top of watering house plants based on specific needs, including the type of plant, amount of sunlight it’s getting, and users local humidity based on their location. This app implements Ruby on Rails for the back end, and VueJS for the front end.

Vue Front End

FEATURES

Features include several rails features, gems and API’s, including HTTParty, OpenWeather API, Twilio API, Figaro, Active Storage, and bcrypt. Front end implements a bootstrap theme, modified for use with Vue for my specific app needs.

Rails Back-End

PLANNED FEATURES

I plan to continue development on this app, as the concept is enjoyable to work with and lends itself to learning new concepts and ideas. The next big step would be to implement a plant store through the rails gem Spree. This could be linked to the original app quite easily once finished.
